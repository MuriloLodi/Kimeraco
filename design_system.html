Motion & Easing

GSAP como motor principal. Todas as animações respeitam prefers-reduced-motion.

/* Easings */ --ease-out: cubic-bezier(0.25, 0.10, 0.25, 1.00); --ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1.00); --ease-expo: cubic-bezier(0.16, 1.00, 0.30, 1.00); /* Transições CSS */ --tr-fast: 150ms var(--ease-out); --tr-med: 300ms var(--ease-out); --tr-slow: 500ms var(--ease-expo);
/* Padrão de reveal por seção */ gsap.from('.card', { opacity: 0, y: 50, scale: 0.97, stagger: { amount: 0.5 }, duration: 0.9, ease: 'expo.out', scrollTrigger: { trigger: '.grid', start: 'top 85%' } });
